Proyecto: ANALISIS CALIDAD DEL AGUA EN COLOMBIA

Descripción:
Desarrollo de un Dashboard en Power BI que permite identificar las ciudades en Colombia con el IRCA más alto y bajo.

Objetivo:
Elaboración de un Dashboard que permite evaluar las ciudades con mayor IRCA en el país, lo cual mide la calidad del agua, data obtenida de datos publicos.

✔ Habilidades aplicadas:

✔ Python (Pandas, Matplotlib, seaborn, Numpy)

✔ EDA - Limpieza y transformación de datos

✔ Visualización de resultados en Power BI

Resultados:
✔ Visualización de manera clara identificando años, ciudades y medición.
✔ Analisis visual que sirve para la toma de decisiones
✔ Analisis de un tema de interés general con aplicación de heramientas de transformación y análisis de datos.
