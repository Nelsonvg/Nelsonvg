Proyecto: FORESIGHT COMERCIAL

Descripción:
Desarrollo de un modelo de pronóstico basado encuatro modelos empleados por Categorias de productos para proyectar sus ventas futuras utilizando datos históricos. Se creó un pipeline en Python para preparar datos, entrenar el modelo y generar predicciones visualizadas, generación de un informe generado por la IA con el analisis de la predicción y comparativo con ventas reales.

Objetivo:
Predecir unidades e ingresos esperados para los próximos 30 días y ayudar a la toma de decisiones en inventario y abastecimiento.

Habilidades aplicadas:

✔ Python (Pandas, Regresión Lineal, Random Forest, Sarima, Matplotlib)

✔ Limpieza y transformación de datos

✔ Modelado de series temporales

✔ Visualización de resultados y generación de informe de la IA.

Resultados:
✔ Modelo con buen ajuste para estacionalidad
✔ Predicciones mensuales por cada una de las categorias
✔ Interpretación visual de tendencias y estacionalidad, metricas de comparación y accuracy del modelo
