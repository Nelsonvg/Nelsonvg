Proyecto: BALANCE SCORE: Dashboard de ventas en Power BI e Informe de la IA.

Descripción:
Construcción de un dashboard dinámico en Power BI que consolida ventas diarias, líneas de venta, Items, Marcas y KPI's de cumplimiento.

Objetivo:
Facilitar la toma de decisiones comerciales y operativas con KPIs claros y visuales.

Habilidades aplicadas:

✔ Power BI: DAX, Power Query, KPIs

✔ Proceso de EDA y ETL en python

✔ Generación de Informe generado por IA mediante el uso de API (LLMS)

✔ Diseño visual orientado a decisiones

Modelado de datos

Resultados:
✔ Dashboard corporativo interactivo replicable a 35 tiendas a nivel nacional.
✔ KPIs: Ventas, Cumplimiento, productos top, unidades.
✔ Actualización y generación de informe cada vez que llega un nuevo archivo
